# 启动命令

## 快速启动（持续重试）

```powershell
Push-Location D:\github_clon\ticket-purchase\damai_appium; python -u -c "from damai_app_v2 import DamaiBot; b=DamaiBot(); b.run_with_retry(max_retries=999999)"; Pop-Location
```

## 快速结束（推荐：先查 PID 再结束）

```powershell
Get-CimInstance Win32_Process | Where-Object { $_.CommandLine -match 'damai_app_v2' } | Select-Object ProcessId, Name, CommandLine | Format-List
Stop-Process -Id <PID> -Force
```

## 一键结束所有抢票进程（谨慎使用）

```powershell
Get-CimInstance Win32_Process | Where-Object { $_.CommandLine -match 'damai_app_v2' } | ForEach-Object { Stop-Process -Id $_.ProcessId -Force }
```

## 可选：前台单次运行（便于排错）

```powershell
Push-Location D:\github_clon\ticket-purchase\damai_appium; python damai_app_v2.py; Pop-Location
```

## 运行前检查（可选）

```powershell
adb devices
appium --version
```
